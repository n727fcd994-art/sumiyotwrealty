  VS Code Live Server 快速安裝指南
═══════════════════════════════════════════════════════

【安裝步驟】

1. 開啟 VS Code

2. 開啟擴充功能面板
   → 按 Ctrl + Shift + X
   → 或點擊左側「擴充功能」圖示

3. 搜尋「Live Server」
   → 找到由 Ritwick Dey 開發的擴充功能

4. 點擊「安裝」按鈕

5. 等待安裝完成

═══════════════════════════════════════════════════════

【使用方法】

1. 在 VS Code 中開啟專案資料夾

2. 開啟 HTML 檔案（如 admin.html）

3. 點擊右下角的「Go Live」按鈕

4. 瀏覽器會自動開啟
   → 網址：http://127.0.0.1:5500/admin.html

═══════════════════════════════════════════════════════

【驗證安裝】

✓ 右下角顯示「Go Live」按鈕
✓ 點擊後瀏覽器自動開啟
✓ 網址是 http://127.0.0.1:5500/ 開頭
✓ 後台系統可以正常載入物件

═══════════════════════════════════════════════════════

【停止伺服器】

點擊右下角「Port: 5500」→ 選擇「Stop Server」
